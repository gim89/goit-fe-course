Строчные елементы

  • a - предназначен для создания ссылок
  • abbr - указывает, что последовательность символов является аббревиатурой
  • area - определяет интерактивную область внутри карты
  • b - устанавливает жирное начертание шрифта
  • br - определяет разрыв строки в тексте
  • bdi - позволяет отображать часть текста в противоположном направлении
  • bdo - устанавливет направление вывода текста: слева направо или справа налево
  • button - определяет нажатие кнопки
  • cite - определяет источник творческой работы
  • code - предназначен для отображения ьекста программного кода
  • col - определяет столбец внутри таблицы
  • data - определяет контент, связанный с машиносчитывающим выходом
  • dd - определяет элемент в списке определений
  • del - определяет текст, который был удален
  • dfn - определяет, где термин определен на веб-странице
  • em - предназначен для акцентрирования текста
  • i - устанавливает курсивное начертание шрифта
  • img - определяет изображение, вставленное на веб-страницу
  • input - определяет интерактивный элемент управления в веб-форме
  • ins - определяет текст, который был вставлен
  • kbd - используется для обозначения текста, который набирается на клавиатуре для названия клавиш
  • label - устанавливает связь междк определенной меткой и элементов формы
  • mark - определяет выделенный текст
  • output - определяет результат вычисления или действия пользователя
  • q - используется для выделения в тексте цитат
  • rp - определяет контент, отображаемый, когда браузер не поддерживает элемент
  • rt - определяет произношение рубиновых аннотаций
  • rtc - определяет текстовый контейнер в рубиновых аннотациях
  • s - отображает текст как перечеркнутый
  • samp - используется для отображения текста, который является результатом вывода компьютерной программы
  • slot - определяет контейнер для разметки пользователя
  • small - уменьшает размер шрифта по сравнению с обычным текстом
  • span - универсальный строчный элемент
  • strong - акцентирует текст, обычно жирным начертанием
  • sub - отображает шрифт в виде нижнего индекса
  • sup - отображает шрифт в виде вернего индекса
  • textarea - создает поле для много строчного текста
  • time - определяет время на 24-часовом такте
  • u - добавляет подчеркивание к тексту
  • var - используется для выделения переменных компьютерных программ
  • wbr - определяет местоположение в блоке текста, где браузер может в конечном итоге вставить разрыв строки

Блочные элементы

  • address - предназначен для хранения информации об авторе
  • article - определяет автономный блок контента, который может существовать в любом контексте. Он может иметь свой собственный заголовок, нижний колонтитул, разделы ... Полезно для списка сообщений в блоге
  • aside - определяет блок содержимого, связанный с основным контентом. Отображается как боковая панель обычно
  • audio - позволяет вставлять аудиоклип в веб-страницу
  • blockquote - предназначен для выделения длинных цитат внутри документа
  • body - кыонтейнер для содержимого веб-страницы. Должен быть прямым дочерним элементом и должен быть предком всех HTML-элементов (кроме тех, которые указаны)
  • canvas - определяет элемент, в котором вы можете рисовать графику
  • caption - определяет заголовок таблицы
  • colgroup - определяет группу столбцов внутри таблицы
  • details - определяет перегружаемый блок контента с сводной информацией и дополнительными сведениями
  • dialog - определяет диалоговое окно, которое можно открыть и закрыть с помощью JavaScript.
  • div - универсальный блочный элемент
  • dl - определяет список определений
  • embed - определяет контейнер для внешнего приложения
  • fieldset - предназначен для группирования элементов формы
  • figcaption - определяет заголовок figure
  • figure - определяет один автономный элемент, обычно изображение
  • footer - определяет нижний колонтитул веб-страницы или раздела
  • form - устанавливает форму на веб-странице
  • h1-h6 - заголовки 1-6 уровней
  • head - определяет контейнер для метаданных веб-страницы
  • header - определяет заголовок веб-страницы или раздела
  • hgroup - определяет заголовок раздела. Он может содержать только уровни заголовков
  • html - определяет корневой элемент документа HTML. Все остальные элементы должны содержаться внутри этого корневого элемента
  • iframe - определяет контейнер для вложенного контекста просмотра: вы можете включить веб-страницу в другую веб-страницу
  • legend - определяет заголовок для содержимого родителя
  • main - определяет основное содержимое веб-страницы. Может отображаться с боковой панелью
  • map - определяет интерактивную карту по изображению
  • meter - определяет горизонтальный метр
  • nav - определяет раздел с навигационными ссылками
  • noframes - определяет контент, отображаемый, когда браузер не поддерживает фреймы
  • noscripts - определяет контент, отображаемый, когда браузер не поддерживает JavaScript
  • object - определяет контейнер для внешнего ресурса
  • ol - определяет упорядоченный список
  • optgroup - определяет группу элементов option
  • p - определяет текстовый абзац
  • pre - определяет блок предварительно форматированного текста
  • progress - определяет индикатор выполнения
  • ruby - определяет рубиновую аннотацию, чтобы показать произношение восточноазиатских символов
  • section - определяет раздел внутри веб-страницы
  • select - определяет выпадающее меню с различными элементами option
  • summary - определяет сводку элемента details
  • table - создает таблицу
  • tbody - определяет группу строк таблицы tr
  • template - определяет местозаполнитель для содержимого, которое может отображаться с использованием JavaScript
  • tfoot - определяет группу строк таблицы tr в конце таблицы table
  • thead - определяет группу строк таблицы tr в начале таблицы table
  • ul - устанавливает маркированный список
  • video - позволяет встроить видеоклип в веб-страницу
